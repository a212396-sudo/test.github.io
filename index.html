Laboratory Exercises

Lab 1: UI Foundations

Exploring fundamental structural mechanisms using declarations in Jetpack Compose UI layout design.

I learned how to use layout box, row and column to make my apps structural integrity efficient. Utilizing these configurations allows me to place visual assets seamlessly.

Lab 2: Interaction & States

Implementing interaction capabilities, event handling listeners, and visual feedback elements.

I make buttons and there also animation to make it smooth when changing values. Handling states successfully creates modern reactive workflows that respond directly to user operations.

Lab 3: Material Design Styling

Applying unified Design Systems with custom palettes, token definitions, and components styling.

I used material theme in my apps. I use it to make my apps theme turn into green suitable with my apps concept, maintaining optimal compliance regarding thematic color identity matches.

Lab 4: Advanced Jetpack Navigation

Constructing robust graphs with dynamic destination flows utilizing standard Jetpack component nodes.

My apps now can go to other screens using navhost and navcontroller, also I saved my fav plant inside favourite page using plantviewmodel to encapsulate global screen memory smoothly.

Lab 5: Room Local Database

Integrating persistent caching architectures using SQLite system mapping query abstractions.

I use room, csv so my data can be safe and process efficiently locally. Offline accessibility data streams provide continuous availability without active server drops dependencies.
